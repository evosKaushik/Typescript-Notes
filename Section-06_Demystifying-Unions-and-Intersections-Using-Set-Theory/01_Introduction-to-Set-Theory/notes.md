# 📝 Set Theory (Mind Map)

## 🌳 Set

- **Collection** → Group of things.
- **Well-defined** → We can clearly decide whether an element belongs to the set or not.

**Example**

```text
A = {1, 2, 3}
```

---

## 📝 Ways to Represent a Set

### 1. Roster Form (Listing Method)

List all elements inside `{}`.

```text
A = {1, 2, 3, 4}
```

### 2. Set-Builder Form

Define a rule or condition.

```text
A = {x : x is a natural number less than 5}
```

Equivalent to:

```text
A = {1, 2, 3, 4}
```

---

## 📦 Types of Sets

- **Finite Set** → Limited number of elements.
  ```text
  A = {1, 2, 3, 4}
  ```

- **Infinite Set** → Endless elements.
  ```text
  N = {1, 2, 3, 4, ...}
  ```

- **Empty (Null) Set** → No elements.
  ```text
  {}
  ```
  Symbol: `∅`

- **Singleton Set** → Exactly one element.
  ```text
  {5}
  ```

---

## 🔢 Standard Sets

| Symbol | Meaning | Example |
|--------|---------|---------|
| **N** | Natural Numbers | `{1, 2, 3, ...}` |
| **W** | Whole Numbers | `{0, 1, 2, ...}` |
| **Z** | Integers | `{..., -2, -1, 0, 1, 2, ...}` |
| **Q** | Rational Numbers | `p/q, q ≠ 0` |
| **R** | Real Numbers | Rational + Irrational |

---

## 🎯 Membership

- `∈` → Belongs to
  ```text
  2 ∈ A
  ```

- `∉` → Does not belong to
  ```text
  5 ∉ A
  ```

---

## 📚 Subsets

- **Subset** → `A ⊆ B`
- **Proper Subset** → `A ⊂ B` (`A ≠ B`)
- **Superset** → `B ⊇ A`
- **Proper Superset** → `B ⊃ A`

Example:

```text
A = {1, 2}
B = {1, 2, 3}
```

```text
A ⊂ B
B ⊃ A
```

---

## 🌍 Universal Set

- Contains **all elements** under consideration.

---

## ⚡ Properties of Sets

- **Order doesn't matter**
  ```text
  {1,2,3} = {3,2,1}
  ```

- **Repetition doesn't matter**
  ```text
  {1,2,2,3,3} = {1,2,3}
  ```

---

## 🎨 Venn Diagram

- **Rectangle** → Universal Set
- **Circle** → Individual Set
- **Points inside** → Elements

Example:

```text
A = {1,2}
B = {1,2,3}

A ⊂ B
```

Circle **A** lies completely inside **B**.
